# 车辆与充电策略（Vehicle）

## 车型信息

- **2024 Hyundai Kona Electric Long Range**

## 品牌充电优先级

Tesla Supercharger → IONITY → Circle K Charge → Recharge

## SOC 电量建议

- 酒店出发（Departure SOC）：90%+
- 午餐充电：40%左右
- 充至（Charge target）：80%
- 抵达酒店（Arrival SOC）：20%~30%
